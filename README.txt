Instructions to run the full application:
1. If ant is already installed then run ant commoand as below
	ant compile jar run
2. The output will be displayed on the console
3. If ant is not installed, then run MainDriverClass.java as a Java Application

MainDriverClass.java
- Is the Test class or driver class

Item.java
- Will have item details

ShoppingCart.java
- will add items into cart, calculates total price and number of items
